# Roadmap — TalentFlow AI

## V1 — MVP (voir 01-cahier-des-charges.md)

Flux complet web : inscription, profils, offres, candidature, score IA de
correspondance, pipeline simple, notifications in-app.

## V1.1

- OAuth Google/GitHub
- Génération de lettre de motivation par IA
- Dashboard entreprise avec quelques graphiques (candidatures/semaine, etc.)

## V2

- Notifications temps réel (Socket.IO)
- Chat recruteur ↔ candidat
- Planification d'entretien (créneaux, calendrier)
- Interface admin dédiée (gestion utilisateurs/entreprises/offres)

## V3

- Application mobile (React Native / Expo)
- Recherche avancée (Elasticsearch) si les volumes le justifient
- Suggestions "meilleurs candidats" pour une offre donnée (IA côté
  entreprise)

## Principe directeur

Ne pas avancer sur une version tant que la version précédente n'est pas
déployée et démontrable de bout en bout. Un MVP qui fonctionne vaut mieux
qu'un V3 à moitié fini.
